## 解决方案

### 方案一：使用GitHub CLI（推荐，最简单）

1. **安装GitHub CLI**：
   - 下载：https://cli.github.com/
   - 或使用包管理器：
     ```powershell
     # Windows (使用Scoop)
     scoop install gh
     
     # 或使用Chocolatey
     choco install gh
     ```

2. **登录GitHub**：
   ```powershell
   gh auth login
   ```
   - 选择 `GitHub.com`
   - 选择 `HTTPS`
   - 选择 `Login with a web browser`
   - 复制显示的验证码
   - 在浏览器中打开显示的链接
   - 输入验证码完成登录

3. **推送代码**：
   ```powershell
   cd C:/Users/Administrator/Desktop/singbox
   git push -u origin main
   ```

### 方案二：配置Git使用代理（如果网络受限）

如果您使用代理，需要配置Git：

```powershell
# 设置HTTP代理（替换为您的代理地址和端口）
git config --global http.proxy http://127.0.0.1:7890
git config --global https.proxy https://127.0.0.1:7890

# 然后推送
git push -u origin main

# 推送完成后，可以取消代理
git config --global --unset http.proxy
git config --global --unset https.proxy
```

### 方案三：使用SSH方式（绕过HTTPS限制）

1. **生成SSH密钥**（如果还没有）：
   ```powershell
   ssh-keygen -t ed25519 -C "your_email@example.com"
   # 按回车使用默认路径
   # 可以设置密码或直接回车
   ```

2. **复制公钥**：
   ```powershell
   cat ~/.ssh/id_ed25519.pub
   # 或
   notepad ~/.ssh/id_ed25519.pub
   ```

3. **添加到GitHub**：
   - 访问：https://github.com/settings/ssh/new
   - Title：填写任意名称（如：我的电脑）
   - Key：粘贴刚才复制的公钥
   - 点击 Add SSH key

4. **修改远程地址**：
   ```powershell
   cd C:/Users/Administrator/Desktop/singbox
   git remote set-url origin git@github.com:moneyfly1/sb.git
   git push -u origin main
   ```
